5.1.修改hive-site.xml

配置以下参数
JDBC连接字符串
name => javax.jdo.option.ConnectionURL
value => jdbc:mysql://localhost:3306/hive?createDatabaseIfNotExist=true&amp;useSSL=false

JDBC的driver
name => javax.jdo.option.ConnectionDriverName
value => com.mysql.jdbc.Driver

username
name => javax.jdo.option.ConnectionUserName
value => hadoop

password
name => javax.jdo.option.ConnectionPassword
value => hadoop

version
name => hive.metastore.schema.verification
value => false

db
name => datanucleus.schema.autoCreateAll
value => true

把${system:java.io.tmpdir}替换成$HIVE_HOME/tmp
把${system:user.name}全部替换成hadoop
